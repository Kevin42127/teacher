## 測試建議

### 推薦測試頁面類型：

1. **系所師資頁面**
   - 格式：`https://大學網址/系所/faculty`
   - 格式：`https://大學網址/系所/staff`
   - 格式：`https://大學網址/系所/people`

2. **教職員介紹頁面**
   - 通常包含姓名、email、科系資訊

3. **系所首頁**
   - 部分系所首頁會列出主要師資

### 測試步驟：

1. 先訪問大學網站首頁
2. 找到「系所」或「學院」連結
3. 進入特定系所網站
4. 尋找「師資介紹」、「教職員」或「Faculty」頁面
5. 複製該頁面的完整 URL 進行測試

## 注意事項

- 請遵守各網站的使用條款
- 建議在非高峰時段進行測試
- 如果某個網站無法訪問，請嘗試其他網站
- 部分網站可能需要等待 JavaScript 載入完成
